== Description ==

Progressive web app let user feel like a natural app on the device, with an immersive user experience. This plugin helps site administrators to quickly enable pwa for their sites just by activating this plugin. 

Read Detail: https://developers.google.com/web/progressive-web-apps/

Features:

* Add to Home Screen Button
* App Icon
* Splash Screen
* Caching

Pro Version:

* Web Push notifications
* Advanced caching
* WooCommerce Support

Requirement:

* SSL certificate must be installed

== Installation ==

= Automatic installation: =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Progressive Web App Basic" and click Search Plugins. Once you've found our plugin you can view details about it such as the the rating and description. Most importantly, of course, you can install it by simply clicking Install Now.

= Manual Installation: =

Unzip the files and upload the folder into your plugins folder (/wp-content/plugins/) overwriting previous versions if they exist
Activate the plugin in your WordPress admin area.
